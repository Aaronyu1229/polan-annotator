# 員工標註指南

> 最後更新：2026-05-01
> 給標註員看，5 分鐘上手第一筆標註。

## 第一次登入

1. 用瀏覽器打開 `https://annotate.dolcenforte.com`
2. 系統會跳到 Cloudflare 登入頁，輸入你的 Gmail
3. 信箱會收到 6 位數 OTP 驗證碼，貼回網頁
4. 進系統，看到音檔清單就成功

OTP 過 30 天會再要一次，每次重登都是一樣的流程。

## 標一筆的流程

1. 首頁清單點任一個還沒打勾的音檔
2. 進到標註頁，按播放鍵聽完整段
3. 拉滑桿給連續維度評分（0–1，每格 0.05）
4. 勾離散維度的 chip（功能角色至少要勾一個，否則無法送出）
5. 填 source type、function roles、style tag 等分類欄位
6. 按 Enter 或畫面下方「儲存並下一筆」

10 個維度全填、source_type 有選、function_roles 至少 1 項，這筆才會被標成「完成」（首頁顯示 ✓）。

## 鍵盤快捷鍵

| 按鍵 | 動作 |
|------|------|
| 空白鍵 | 播放 / 暫停音訊 |
| Enter | 儲存目前標註並跳下一筆 |
| Cmd+S | 強制存檔（不跳下一筆） |

> 注意：游標在 textarea 時空白鍵會輸入空白、不會觸發播放。要先 Tab 出去或點別的地方。

## 標到一半暫停？

- 系統每 3 秒沒操作會自動把當前狀態存到瀏覽器（draft），**不會送到伺服器**
- 下次打開同一筆會跳 modal 問「繼續草稿還是重新開始」
- 真的要保存，按一次 Cmd+S 或 Enter，這樣才會寫進資料庫

換瀏覽器 / 換電腦 draft 不會跟著走，重要的還是要按 Enter 確認。

## ⚠️ 4 個尚未確認的維度

下面 4 個維度的定義文字是 Aaron 推敲的，Amber 還在試標驗收，UI 上會看到 ⚠️ 圖示：

- Emotional Warmth（情緒溫暖度）
- Tension Direction（張力走向）
- Event Significance（事件重要性）
- World Immersion（世界沉浸感）

**標的時候若覺得定義模糊，不要硬猜。** 點該維度旁的 💬 icon 留下反饋（描述你的疑慮即可），Amber 會看到並調整定義文字。硬猜會污染後續校準資料。

## 出問題該怎麼辦

| 情境 | 立刻試的解法 |
|------|------|
| 收不到 OTP | 看垃圾信夾、確認你給 Amber 的 Gmail 沒拼錯 |
| 進系統後白屏 | 重整一次；若 19 秒後恢復，是剛好遇到部署 |
| 上傳 / 拖滑桿卡住 | 換 Chrome / Edge，Safari 的 audio decode 有時候會出怪事 |
| 標完按 Enter 沒反應 | 檢查 function_roles 是否至少勾了一個 |

還沒解掉 → LINE Amber，她會轉給 Aaron。**不要自己嘗試重灌或改設定**。
